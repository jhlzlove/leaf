这个框架中浪子没有使用很多流行的 lib，一个简单的项目框架，使用什么工具交给使用者自己去选择，这样可以选择自己趁手的工具，效率、正确率等等都会大幅提高！所以这个项目很少引入其它工具库，基本都是使用

对于其它的、好用的、流行的第三方库的使用权就交给用户选择吧。毕竟 Java 生态很庞大，好用的库有很多，根据自己或者团队选择合适的就可以。
