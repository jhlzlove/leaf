## 数据库简单设计

id 字段预留出来，不同业务可以采用不同的 id 生成策略算法
如果数据库字段为整型数字，那么默认 0 统一都是正常情况

### 一、表字段

#### 1. 通用字段

除关联表以外的其它表基本都含有这几个字段，不过也应视具体情况而定。

| 名称     | 数据库字段  | 数据库类型   | 实体字段   | 实体类型      | 说明 |
| -------- | ----------- | ------------ | ---------- | ------------- | ---- |
| 创建者   | create_by   | varcahr(64)  | createBy   | String        |      |
| 更新者   | update_by   | varchar(64)  | updateBy   | String        |      |
| 创建时间 | create_time | datetime     | createTime | LocalDateTime |      |
| 更新时间 | update_time | datetime     | updateTime | LocalDateTime |      |
| 备注     | remark      | varchar(255) | remark     | String        |      |
